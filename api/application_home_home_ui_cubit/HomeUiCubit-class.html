HomeUiCubit class

Cubit responsible for managing the home screen UI state and operations.

This cubit handles core home functionality including:

  • Home details fetching and management
  • User location tracking and updates
  • Coin balance management and updates
  • Platform polling functionality
  • Navigation state management (drawer, tabs)
  • User blacklist checking
  • Notification status management
  • One-time purchase handling

The cubit integrates with location services, chat functionality, and provides real-time updates for user-specific data.
