PickWinnerBloc

PickWinnerBloc class

BLoC responsible for managing the winner selection process for giveaways and campaigns.

This is one of the core business logic components that handles:

  • Giveaway creation and management
  • Comment fetching from social media platforms (Instagram, YouTube, TikTok)
  • Comment filtering based on advanced criteria (keywords, mentions, likes, etc.)
  • Random winner selection with backup winners
  • Video recording of the selection process
  • Certificate generation for winners
  • Progress tracking and user feedback during the process
  • Integration with multiple other blocs for comprehensive giveaway management

The bloc supports both campaign-based and standalone giveaway formats, with sophisticated filtering algorithms to ensure fair and valid winner selection.
