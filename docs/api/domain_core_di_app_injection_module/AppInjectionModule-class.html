AppInjectionModule

AppInjectionModule class abstract

Dependency injection module for app-wide services and utilities.

This module provides all the core dependencies needed throughout the app including external services, platform-specific implementations, and third-party integrations. It uses the injectable package for automatic dependency injection setup.
