Bingo Flutter App

A comprehensive Flutter application for managing bingo campaigns, giveaways, and social media interactions. This app provides a complete platform for influencers and content creators to create, manage, and track their bingo campaigns across various social media platforms.

📱 Overview

Bingo is a feature-rich mobile application built with Flutter that enables users to:

  • Create and manage bingo campaigns
  • Conduct giveaways and contests
  • Track campaign performance and analytics
  • Interact with followers through chat and reactions
  • Manage coins and rewards system
  • Generate certificates for winners
  • Handle video editing and media management

🏗️ Architecture

The application follows Clean Architecture principles with a clear separation of concerns:

lib/
├── application/          # Business logic layer (BLoC/Cubit)
├── domain/              # Domain entities and interfaces
├── infrastructure/      # Data sources and external services
└── presentation/        # UI layer (Screens and Widgets)

Architecture Layers

  1. Presentation Layer (lib/presentation/)

    • UI screens and widgets
    • State management using BLoC pattern
    • User interactions and navigation

  2. Application Layer (lib/application/)

    • Business logic implementation
    • BLoC/Cubit state management
    • Use cases and application services

  3. Domain Layer (lib/domain/)

    • Core business entities
    • Repository interfaces
    • Domain services and use cases

  4. Infrastructure Layer (lib/infrastructure/)

    • Data sources implementation
    • External API integrations
    • Local storage and caching

🚀 Getting Started

Prerequisites

  • Flutter SDK (^3.5.3)
  • Dart SDK
  • Android Studio / VS Code
  • Git

Installation

  1. Clone the repository

    git clone <repository-url>
cd bingo-flutter-app

  2. Install dependencies

    flutter pub get

  3. Configure Firebase

    • Add your google-services.json to android/app/
    • Add your GoogleService-Info.plist to ios/Runner/

  4. Configure environment variables

    • Create a .env file in the root directory
    • Add necessary API keys and configuration

  5. Run the application

    flutter run

Build for Production

Android:

flutter build apk --release

iOS:

flutter build ios --release

  3. Add the configuration files to respective platform folders

Configure API endpoints and keys in the environment configuration files.
