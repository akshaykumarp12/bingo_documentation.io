IAuthenticationImpl

IAuthenticationImpl class

Implementation of IAuthenticationFacade for authentication operations.

This class provides concrete implementation of all authentication-related operations including user registration, login, profile management, password recovery, and social authentication validation.

It handles HTTP requests through DioClient, error handling through RunSafely, and local storage through SharedPreferences.
